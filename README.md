# hotel-booking

## 功能
使用者:
- 可以選擇時間預定房間、正餐、活動
- 可以觀看目前所有的預訂紀錄

管理面板:
- 可以設定房間、正餐、活動
- 可以檢視預訂清單，並將結果匯出成 PDF
- 可以檢視當前項目剩餘數量
